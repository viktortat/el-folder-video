# Если что-то не работает

## Видео есть в списке, но не воспроизводится

Расширение файла поддерживается, но Chromium может не уметь декодировать конкретный кодек. Откройте файл в системном плеере через кнопку в верхней панели. Если там ролик работает, перекодируйте его в совместимый формат и снова откройте папку.

## Не появляется ускоренная копия

Проверьте, что FFmpeg установлен и доступен по команде `ffmpeg` через `PATH`. После добавления FFmpeg в `PATH` перезапустите Folder-video. Ускоренная копия создаётся рядом с исходным роликом с суффиксом `_2x`.

## В списке не видно вложенных видео

Включите `Recursive` и повторите сканирование. Символические ссылки и недоступные каталоги приложение пропускает, чтобы не остановить просмотр всей папки.

## Миниатюры появляются не сразу

Ленты кадров создаются по очереди. Откройте нужную страницу и подождите: задачи активной страницы получают приоритет.
