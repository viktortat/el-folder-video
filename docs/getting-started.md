# Первый запуск

После запуска выберите папку с видео. Folder-video не открывает прошлую папку автоматически, поэтому новый сеанс начинается с выбора каталога.

## Откройте папку

1. Нажмите кнопку выбора папки.
2. Укажите каталог с видео.
3. Включите `Recursive`, если нужно искать во вложенных папках.

Папку также можно перетащить в окно. В библиотеке появятся имя файла, длительность, размер, дата изменения и лента кадров.

Чтобы увидеть новые файлы, нажмите кнопку обновления рядом с путём папки.

## Ориентируйтесь в библиотеке

- Кнопка с домиком возвращает на главную страницу.
- Повторный клик по уже открытому ролику активирует его вкладку, а не создаёт вторую.
- Поле «Фильтр файлов» ищет фрагмент имени без учёта регистра.
- Сортировка по имени или дате и постраничная навигация помогают в больших папках.

## Настройте просмотр

Кнопка с шестерёнкой открывает вкладку настроек. Там можно выбрать число колонок, шаг кадров, автопрокрутку и состояние панелей. Изменения вступают в силу после нажатия «Сохранить».

Подробности есть в [настройках](features/settings.md). Дальше откройте [библиотеку видео](features/video-library.md) или сразу перейдите к [покадровому просмотру](features/video-review.md).
