# Folder-video: пользовательская документация

Когда в папке лежат десятки или сотни роликов, обычный Проводник помогает найти файл по имени, но почти не помогает понять, что внутри. Folder-video показывает видео лентами кадров, а нужный момент открывает в отдельной вкладке с точной сеткой миниатюр.

![Список видео с лентой кадров](screens/video-library.webp)

## Что вы получите

- Быстрый обзор локальной видеопапки без импорта файлов в приложение.
- Поиск ролика по части имени, сортировку по имени или дате и постраничную навигацию.
- Точный переход к сцене: кликните по кадру или перетащите маркер по сетке.

## Кому пригодится Folder-video

Приложение удобно, если вы:

- разбираете архив записей экрана, лекций, интервью или рабочих материалов;
- выбираете нужный фрагмент по изображению, а не по длинному имени файла;
- хотите хранить рядом с видео ссылки, описание и несколько тегов.

## Требования и ограничения

- Основная платформа для полного набора функций - Windows 10/11.
- Поддерживаются `mp4`, `webm`, `mov`, `avi`, `mkv`, `m4v` и `ogv`.
- Приложение работает с локальными файлами и не загружает их в интернет.
- Для ускорения ролика в два раза нужен установленный FFmpeg, доступный по имени `ffmpeg` в системном `PATH`.
- Контекстное меню Проводника, перенос файлов и корзина Windows относятся к Windows-сценариям.
