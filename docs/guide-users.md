# Folder-video: пользовательская документация

Когда в папке много роликов, имя файла редко помогает быстро вспомнить нужную сцену. Folder-video показывает ленты кадров по всей длине видео и открывает выбранный ролик в отдельной вкладке для детального просмотра.

![Библиотека видео с лентами кадров](screens/user-library.png)

## Что получится

- Найти ролик по изображению, фильтру или дате, не открывая файлы по очереди.
- Перейти к нужному моменту через сетку кадров, клик или перетаскивание маркера.
- Сохранить для ролика заметки, ссылки и теги, чтобы вернуться к нему позже.

## Требования и ограничения

- Полный набор возможностей рассчитан на Windows 10 и Windows 11.
- Поддерживаются `mp4`, `webm`, `mov`, `avi`, `mkv`, `m4v` и `ogv`.
- Приложение работает с локальными файлами и не загружает содержимое роликов в интернет.
- Для создания ускоренной копии ролика нужен FFmpeg, доступный как `ffmpeg` через `PATH`.
- Воспроизведение зависит от кодека, который поддерживает Chromium. Подходящее расширение само по себе не гарантирует воспроизведение.

Если видео не открывается или ускорение не запускается, смотрите [решение частых проблем](help/troubleshooting.md).
